# Liminal Horror Reel Generator

Automated tool to create "Choose Your Door" liminal horror content for TikTok/Instagram.

## Quick Start

1.  `pip install -r requirements.txt`
2.  `python3 main.py`

## Features

-   **AI Image Generation**: Uses DALL-E 3 (via OpenAI API).
-   **Video Composition**: Automates editing with MoviePy.
-   **Upload Hooks**: Placeholder system for social media automation.

## Configuration

Check `.env` for API keys and Mock Mode settings.
